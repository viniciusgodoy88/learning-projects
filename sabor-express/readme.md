### 🍽️ Sabor Express API
Primeira API desenvolvida em Python, criada como parte do meu processo de aprendizado em back-end.
Este projeto foi construído com base no curso:

### 📚 Alura – Python: Crie a sua primeira aplicação
O objetivo foi aplicar conceitos fundamentais de Python na construção de uma aplicação estruturada, evoluindo para o desenvolvimento de uma API simples.

#### 📌 Sobre o Projeto
O Sabor Express é uma aplicação que simula o gerenciamento de restaurantes, permitindo:
Cadastro de restaurantes
Listagem de restaurantes
Alteração de status (ativo/inativo)
Organização da lógica em funções
Estruturação de código de forma modular
Este projeto representa minha transição do aprendizado básico de Python para a construção de aplicações back-end.
