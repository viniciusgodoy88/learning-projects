📘 Learning Projects

Este repositório contém uma coleção de pequenos projetos em Python criados para praticar lógica de programação, estruturas de controle, operações matemáticas e manipulação de dados. Cada script é uma aplicação independente com foco em fundamentos da linguagem e resolução de problemas.

🧠 Projetos & Funcionalidades
📊 Calculadoras & Cálculos
Arquivo	Descrição
bmicalculator.py	Calculadora de IMC (Índice de Massa Corporal) que lê peso e altura e exibe o resultado com classificação de peso.
simplecalculator.py	Calculadora básica para operações aritméticas (adição, subtração, multiplicação e divisão).
calculatorversion2.py	Versão estendida da calculadora com mais opções ou talvez interface aprimorada.
toolcalculator.py	Calculadora com ferramentas adicionais (ex.: porcentagem, raiz, etc.).
loancalculator.py	Calculadora de empréstimo que pode estimar parcelas, juros e prazos.
montlybudget.py	Calculadora de orçamento mensal para controle de receitas e despesas.
projectcalculating.py	Script para cálculos gerais relacionados a projetos (ex.: custo, tempo ou métricas).
🔍 Lógica & Controle
Arquivo	Descrição
logiclist.py	Exercício de lógica manipulando listas e realizando operações como ordenação e filtragem.
numberparity.py	Verifica se um número é par ou ímpar e exibe o resultado.
temperaturealert.py	Sistema simples de alerta baseado na temperatura informada.
studentgrade.py	Calcula e exibe a média de um aluno e sua situação (aprovado/reprovado).
officecontrol.py	Controle de fluxos de trabalho ou tarefas de escritório; pode incluir lógica de organização ou cadastro.
sellfruits.py	Simula um sistema de venda de frutas com cálculo de preço total.
🛠 Tecnologias Usadas

🐍 Python – Principal linguagem do repositório

💻 Conceitos aplicados: lógica de programação, condicionais, loops, funções, entrada e saída de dados

🧪 Projetos pequenos e focados em aprendizado prático

📁 Organização simples por script para facilitar o estudo individual de cada tema
