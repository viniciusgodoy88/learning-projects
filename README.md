📘 Learning Projects

Este repositório reúne uma coleção de projetos práticos em Python desenvolvidos com foco no fortalecimento da lógica de programação, resolução de problemas e consolidação dos fundamentos da linguagem.

Cada script representa um exercício aplicado a situações reais, como cálculos financeiros, validações, simulações de vendas e controle de dados.

🎯 Objetivo: Evolução técnica contínua e prática estruturada de fundamentos essenciais para desenvolvimento de software.
🧠 Objetivos de Aprendizado
Este repositório foi criado para praticar:
Estruturas condicionais (if, elif, else)
Estruturas de repetição (for, while)
Manipulação de listas
Entrada e saída de dados
Operações matemáticas
Organização de código
Boas práticas de legibilidade
Raciocínio lógico aplicado

1️⃣ bmicalculator.py – Calculadora de IMC
📌 Descrição
Calcula o Índice de Massa Corporal (IMC) com base no peso e altura informados pelo usuário.

⚙️ Funcionalidades
Entrada de peso e altura
Cálculo automático do IMC

Classificação do resultado:
Abaixo do peso
Peso normal
Sobrepeso
Obesidade
Exibição formatada do resultado

💡 Conceitos aplicados
Operações matemáticas
Estruturas condicionais
Validação básica de entrada
